## T2.1 — Implementar `show_worktree_error`

**Onde:** `repo-launcher.sh` — adicionar função após `resolve_worktree` (~linha 486).

**Signature:**

```bash
show_worktree_error() {
  local repo="$1"
  local branch="$2"
  local action_type="$3"   # local | remote | new
  local error_msg="$4"     # geralmente RESOLVE_FULL_ERROR
  ...
}
```

**Layout do output (`tput cols` dinâmico):**

```
 ERROR   worktree add failed
 ──────────────────────────────────────────

 repo     <repo>
 branch   <branch>
 action   <type>

 git error:
   <error wrapeado em (cols-4) com fold -s>

 what you can do:
   <sugestão contextual wrapeada>

 press any key to continue
```

**Cores** (truecolor inline, mesma convenção do resto do arquivo):
- header `ERROR` em vermelho `#f38ba8`
- separador em sep `#585b70`
- labels `repo`/`branch`/`action`/`git error:`/`what you can do:` em dim `#6c7086`
- valores em text `#cdd6f4`
- `press any key` em dim

**Wrap dinâmico:**

```bash
local cols
cols=$(tput cols 2>/dev/null || echo 80)
local wrap_width=$((cols - 4))
((wrap_width < 30)) && wrap_width=30   # piso de segurança

# Wrap do erro (preservando palavras com -s)
printf '%s\n' "$error_msg" | fold -s -w "$wrap_width" | sed "s/^/   /"
```

**Pattern matching para sugestões:**

```bash
local hint
case "$error_msg" in
  *"is already used by worktree at"*)
    hint="branch já está em uso em outro worktree. Feche aquele primeiro ou escolha outra branch."
    ;;
  *"branch named"*"already exists"*|*"a branch named"*"already exists"*)
    hint="branch local já existe. Volte e escolha tipo 'local' em vez de 'new'."
    ;;
  *"not a valid branch name"*|*"is not a valid branch name"*)
    hint="nome inválido. Evite trailing dot, leading dash, ou chars especiais (: \\ * ? < > | \")."
    ;;
  *"already exists"*)
    hint="diretório do worktree já existe (.wt/${branch//\//--}). Remova manualmente se for órfão."
    ;;
  *)
    hint="log completo em ~/.local/state/wezterm/worktree.log"
    ;;
esac

printf '%s\n' "$hint" | fold -s -w "$wrap_width" | sed "s/^/   /"
```

**Dismissão:**

```bash
clear
# desenhar tudo
# ...
read -rsn1 _ < /dev/tty
clear
```

(Usar `< /dev/tty` para garantir que o read pegue input do terminal, mesmo se a função for chamada de dentro de um pipeline ou subshell.)
